## 1. 产品概述
RS-485总线通信模拟器，用于可视化展示多节点通过RS-485半双工总线发送数据时的冲突检测与地址优先级仲裁机制。
- 主要用途：工业通信教学演示、协议分析、嵌入式开发调试
- 目标用户：嵌入式工程师、通信协议学习者、高校师生

## 2. 核心功能

### 2.1 功能模块
1. **模拟器主页**：节点控制面板、总线示波器、状态日志、仲裁结果展示

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 模拟器主页 | 节点控制面板 | 添加/删除节点，设置节点地址和发送数据，触发发送 |
| 模拟器主页 | 总线示波器 | 实时显示各节点TX/RX波形、总线实际波形、冲突标记 |
| 模拟器主页 | 状态日志 | 记录总线事件：发送开始、冲突检测、仲裁结果、发送完成 |
| 模拟器主页 | 仲裁结果展示 | 显示冲突时各节点的优先级排序、获胜节点 |

## 3. 核心流程
用户添加多个节点，设置不同地址和数据，点击发送触发冲突场景，模拟器执行位级仲裁算法，根据地址优先级决定获胜节点，在示波器上可视化冲突过程和最终总线波形。

```mermaid
flowchart TD
    "A[添加节点]" --> "B[设置节点地址与数据]"
    "B" --> "C[触发多节点同时发送]"
    "C" --> "D[位级逐位仲裁]"
    "D" --> "E{"是否冲突?"}"
    "E" --> "F[是: 低地址节点获胜]"
    "E" --> "G[否: 正常发送]"
    "F" --> "H[总线波形更新]"
    "G" --> "H"
    "H" --> "I[日志记录与状态更新]"
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深色工业风，深蓝灰背景 (#0f1623)，青色强调 (#00d4ff)
- 辅色调：成功绿 (#10b981)，警告黄 (#f59e0b)，冲突红 (#ef4444)
- 字体：JetBrains Mono（等宽字体，适合显示数据和波形）
- 布局：三栏布局 - 左侧节点控制、中间总线示波器、右侧状态日志
- 按钮风格：圆角矩形，深色背景+青色边框，悬停时青色高亮

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|---------------------|
| 模拟器主页 | 节点控制面板 | 卡片式节点列表、地址输入框、数据输入框、发送按钮 |
| 模拟器主页 | 总线示波器 | Canvas波形图、时间轴、通道标签、冲突高亮 |
| 模拟器主页 | 状态日志 | 滚动日志列表、时间戳、彩色事件标签 |
| 模拟器主页 | 仲裁结果展示 | 优先级排序表、获胜节点高亮卡片 |

### 4.3 响应式设计
桌面端优先，支持平板端自适应，最小宽度1024px
