## 1. 产品概述

布尔函数化简工具，基于Quine-McCluskey（QMC）算法，帮助数字电路设计人员和学生快速将复杂的布尔函数化简为最简与或式。支持真值表和积和式两种输入方式，可视化展示化简过程和结果。

- 核心用途：数字逻辑电路设计、布尔代数教学辅助、逻辑表达式化简
- 目标用户：电子工程专业学生、数字电路设计师、计算机科学学习者
- 产品价值：替代手工卡诺图化简，支持多变量（最多8个变量）复杂函数的精确化简

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 使用所有化简功能，查看历史记录（本地存储） |

### 2.2 功能模块
1. **首页**：输入模式选择、变量数配置、主操作区
2. **真值表输入**：动态生成真值表，点击选择输出值
3. **积和式输入**：直接输入最小项编号，支持无关项
4. **结果展示**：最简与或式输出、化简步骤、卡诺图可视化

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|----------|----------|
| 首页 | 模式切换 | 在真值表输入和积和式输入两种模式间切换 |
| 首页 | 变量配置 | 支持2-8个布尔变量的配置，动态调整输入界面 |
| 真值表页面 | 真值表交互 | 点击输出列切换0/1/×（无关项），支持全选/反选 |
| 积和式页面 | 表达式输入 | 输入最小项编号，用逗号分隔，无关项用d开头 |
| 结果展示 | 公式渲染 | 使用数学符号展示最简与或式，支持复制 |
| 结果展示 | 化简步骤 | 展示QMC算法的关键步骤：素项表生成、本质素项选择 |

## 3. 核心流程

用户选择输入模式 → 配置变量数量 → 输入真值表或积和式 → 点击化简按钮 → 后端执行QMC算法 → 前端展示最简表达式和化简过程

    D --> E["后端QMC算法计算"]

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深靛蓝 (#1e3a5f)，代表严谨和科技感
- **辅助色**：琥珀橙 (#f59e0b) 用于强调操作和结果
- **中性色**：深灰背景 (#0f172a)、浅灰文字 (#e2e8f0)
- **按钮风格**：圆角8px，悬停时有微妙的发光效果
- **字体**：标题使用 "Space Mono" 等宽字体，正文使用 "Inter"
- **布局风格**：卡片式布局，带细边框和柔和阴影，深色主题
- **图标风格**：使用 lucide-react 线性图标，统一线条粗细

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|---------|----------|--------|
| 首页 | Header | 应用标题、副标题、暗色主题 |
| 首页 | 配置区 | 变量数滑块、模式切换标签 |
| 真值表页面 | 表格区 | 动态生成的真值表，可点击的输出单元格 |
| 积和式页面 | 输入区 | 文本输入框、示例提示、格式说明 |
| 结果展示 | 结果卡 | 大号字体的最简表达式、复制按钮 |
| 结果展示 | 步骤区 | 可折叠的化简步骤面板 |

### 4.3 响应式
- Desktop-first 设计，断点 768px
- 移动端：表格横向滚动，按钮全宽，单列布局
- 桌面端：双列布局，左侧输入区，右侧结果区
- 触摸优化：增加点击区域，支持滑动切换模式

### 4.4 视觉细节
- 背景使用微妙的网格纹理和径向渐变
- 真值表单元格悬停时有缩放和高亮效果
- 化简结果出现时有淡入动画
- 公式使用 LaTeX 风格渲染（非标准符号显示：A' 表示非 A）
- 无关项使用特殊的紫色背景区分
