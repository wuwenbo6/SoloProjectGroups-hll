## 1. 产品概述

OTN帧模拟器是一款面向光传输网络（OTN）工程师和教学场景的专业仿真工具，支持高阶ODUk（ODU2/ODU3）与低阶ODU0的复用/解复用过程模拟，提供映射开销调整与时隙占用可视化功能。

- 帮助用户直观理解OTN帧结构、ODUk复用层级与映射开销机制
- 目标用户：光通信工程师、网络运维人员、通信专业师生

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用方式 | 核心权限 |
|------|----------|----------|
| 操作员 | 直接访问 | 执行复用/解复用操作、调整参数、查看可视化 |

### 2.2 功能模块

1. **模拟器主页面**：OTN帧结构可视化、ODUk复用/解复用操作、映射开销编辑、时隙占用展示
2. **帧详情页面**：单帧结构展开查看、各区域字节级详情、开销字段解析

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 模拟器主页面 | 帧结构概览 | 展示ODU2/ODU3帧的4行×3824列结构，分区着色显示FAS/MFAS/ODUk OH/OPUk OH/Payload/FEC区域 |
| 模拟器主页面 | 复用/解复用控制 | 支持将ODU0信号映射入ODU2的指定时隙，支持从ODU2中解复用提取ODU0，GMP/AMP映射方式切换 |
| 模拟器主页面 | 映射开销编辑 | 可视化编辑ODUk OH（PM/TCM/APS等）和OPUk OH（PT/PSI/Jo/jc等）字段，实时反馈合法性校验 |
| 模拟器主页面 | 时隙占用图 | 以时隙矩阵/甘特图形式展示ODU2的8个TS（1.25Gbps）被ODU0占用的状态，支持拖拽分配 |
| 帧详情页面 | 字节级查看 | 点击帧结构区域展开，显示每个字节的含义和当前值 |
| 帧详情页面 | 开销解析 | 对开销字段进行协议级解析，显示各bit/byte含义 |

## 3. 核心流程

用户操作流程：选择高阶ODU类型 → 配置低阶ODU0信号参数 → 执行复用映射 → 查看时隙占用结果 → 可选调整映射开销 → 执行解复用验证

```mermaid
flowchart TD
    "A[选择高阶ODU类型<br/>ODU2/ODU3]" --> "B[配置低阶ODU0信号<br/>数量与速率]"
    "B" --> "C[选择映射方式<br/>GMP/AMP]"
    "C" --> "D[执行复用映射"
    "D" --> "E[查看时隙占用图"
    "E" --> "F{调整映射开销?}"
    "F" --> "是" --> "G[编辑开销字段"
    "G" --> "E"
    "F" --> "否" --> "H[执行解复用"
    "H" --> "I[验证原始信号"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝色（#0A1628）象征光纤通信，辅以青色（#00D4FF）高亮信号流转
- 次要色：琥珀色（#FFB800）表示告警/占用，翠绿色（#00FF88）表示空闲/正常
- 按钮风格：圆角微3D效果，悬停有发光边框
- 字体：显示字体 Space Grotesk，数据字体 JetBrains Mono
- 布局风格：深色主题仪表盘布局，左侧控制面板 + 中央帧可视化 + 右侧时隙图
- 图标风格：线性科技风（lucide-react）

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 模拟器主页面 | 帧结构概览 | 深色画布，4行×3824列像素级帧结构图，区域用渐变色区分，悬停显示字节信息tooltip |
| 模拟器主页面 | 复用/解复用控制 | 左侧面板：ODU类型下拉选择、映射方式Radio、ODU0信号列表（可添加/删除）、复用/解复用按钮带动画 |
| 模拟器主页面 | 映射开销编辑 | 弹出式侧边抽屉：表单化编辑各开销字段，分组显示PM/TCM/APS/PT/PSI，实时校验红色标记 |
| 模拟器主页面 | 时隙占用图 | 右侧面板：8个时隙横条（ODU2），每个可拖拽ODU0到空闲时隙分配，颜色编码标识占用/空闲 |
| 帧详情页面 | 字节级查看 | 放大区域网格，每个字节单元格可点击，显示协议字段名和当前值 |
| 帧详情页面 | 开销解析 | 树形展开结构，bit级标注含义 |

### 4.3 响应式设计

- 桌面优先（1920×1080最佳），适配1366×768以上分辨率
- 三栏布局在小屏下可折叠为标签页切换

### 4.4 3D场景指引

不适用
