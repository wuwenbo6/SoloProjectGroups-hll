# LC3 音频压缩对比工具

一个基于Electron的桌面应用，用于演示LC3（Low Complexity Communication Codec）音频编解码的压缩效果，并提供主观质量评估（MOS分）。

## 功能特性

- 🎵 **WAV文件上传**：支持拖拽或点击上传16位PCM格式的WAV音频文件
- 🔧 **LC3编解码**：基于MDCT变换的LC3风格音频压缩算法
- 🎚️ **可调码率**：支持8-160 kbps的码率调节
- 📊 **MOS分评估**：基于PESQ算法原理的主观质量评分（1-5分）
- 🎧 **音频对比**：原始音频与压缩后音频的并列播放对比
- 📈 **量化指标**：显示压缩率、SNR、对数谱距离等客观指标

## 使用说明

1. **上传WAV文件**：点击上传区域或拖拽WAV文件到界面
2. **设置码率**：拖动滑块选择目标编码码率（8-160 kbps）
3. **开始处理**：点击"开始压缩处理"按钮
4. **查看结果**：
   - 查看MOS分和质量评估
   - 对比原始音频和压缩后音频
   - 查看压缩率和其他技术指标

## 技术实现

### LC3编码器

- 使用10ms帧长进行分帧处理
- 基于MDCT（改进离散余弦变换）的频域转换
- 自适应量化（根据目标码率调整量化精度）
- 重叠相加（OLA）的帧重建

### MOS分计算

- 分段信噪比（SegSNR）
- 对数谱距离（LSD）
- 全局信噪比（Global SNR）
- 基于PESQ映射的主观质量评分

## 技术栈

- **Electron 30**：跨平台桌面应用框架
- **WaveFile**：WAV文件解析与生成
- **HTML5 Audio**：音频播放
- **原生JavaScript**：编解码算法实现

## 注意事项

- 目前仅支持16位PCM格式的WAV文件
- 长音频文件处理可能需要较长时间
- 这是一个演示版本的LC3实现，如需生产环境使用，建议集成官方liblc3库
