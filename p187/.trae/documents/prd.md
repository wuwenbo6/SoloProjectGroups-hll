## 1. 产品概述

UWB测距数据卡尔曼滤波分析平台，用于处理超宽带（UWB）测距传感器采集的带时间戳数据，通过卡尔曼滤波算法降低多径效应噪声，直观展示滤波前后的数据对比效果。

- 核心用途：UWB测距数据降噪处理与可视化分析
- 解决问题：UWB信号多径传播导致的测距误差与噪声
- 目标用户：物联网工程师、定位系统开发者、科研人员
- 产品价值：提供直观的降噪效果验证工具，加速UWB定位系统调试

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 上传数据、执行滤波、查看对比图表、下载结果 |

### 2.2 功能模块
1. **主页/数据处理页**：文件上传区域、滤波参数配置、处理结果展示
2. **数据可视化模块**：滤波前后折线对比图、统计指标卡片
3. **数据导出模块**：支持下载滤波后的数据文件

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|----------|----------|
| 主页面 | 文件上传区 | 支持拖拽或点击上传CSV/JSON格式的UWB测距数据，实时显示上传进度 |
| 主页面 | 参数配置区 | 卡尔曼滤波参数可调（过程噪声、测量噪声、初始估计误差） |
| 主页面 | 数据预览区 | 展示上传数据的前10条记录，包含时间戳、原始距离字段 |
| 主页面 | 对比图表区 | 双折线图展示原始数据与滤波后数据的对比，支持缩放、悬停查看数值 |
| 主页面 | 统计指标区 | 展示均值、方差、峰值噪声、降噪比等关键指标 |
| 主页面 | 数据导出区 | 支持下载滤波后的数据为CSV格式 |

## 3. 核心流程

用户上传UWB测距数据文件 → 系统解析并预览数据 → 用户配置卡尔曼滤波参数 → 点击执行滤波 → 后端应用卡尔曼滤波算法 → 返回滤波后数据 → 前端渲染对比图表与统计指标 → 用户可下载处理结果

```mermaid

    A["上传数据文件"] --> B["解析与预览"]
    B --> C["配置滤波参数"]
    C --> D["执行卡尔曼滤波"]
    D --> E["生成对比图表"]
    E --> F["展示统计指标"]
    F --> G["导出结果数据"]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深空蓝 #0F172A（背景）、科技青 #06B6D4（强调色）、警示橙 #F97316（辅助色）
- **按钮风格**：圆角8px，微渐变填充，悬停有发光效果
- **字体**：显示字体使用 Space Mono（等宽科技感），正文字体使用 Inter
- **布局风格**：卡片式布局，深色主题，网格对齐，数据可视化区域占据主要视觉空间
- **图标风格**：使用 lucide-react 线性图标，科技感风格

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|---------|----------|--------|
| 主页面 | 顶部导航 | Logo、项目名称、主题切换、示例数据按钮 |
| 主页面 | 上传区域 | 虚线边框拖拽区、文件图标、格式提示、进度条 |
| 主页面 | 参数卡片 | 滑块控件、数值输入框、参数说明tooltip |
| 主页面 | 图表区域 | 双折线对比图、图例、工具栏（缩放/重置/下载图片） |
| 主页面 | 指标卡片 | 4个统计卡片网格，数值+变化率显示 |
| 主页面 | 数据表格 | 固定高度可滚动表格，时间戳与距离数据分列 |

### 4.3 响应式
- 桌面端优先（1280px+）：两栏布局，左侧参数配置+数据预览，右侧图表展示
- 平板端（768px-1279px）：上下堆叠布局，图表区域自适应宽度
- 移动端（<768px）：单列布局，简化参数配置，优化触控区域

### 4.4 动效设计
- 页面加载：元素淡入+上移动画，错开0.1s延迟
- 文件上传：拖拽进入时边框高亮+背景呼吸效果
- 数据处理：加载状态使用脉冲动画，结果展示使用渐入效果
- 图表交互：悬停数据点时放大高亮，平滑过渡动画
