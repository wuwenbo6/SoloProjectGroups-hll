## 1. 产品概述

路测数据热力图分析平台——上传路测CSV文件（含经纬度+RSRP/SINR），通过反距离加权（IDW）插值生成热力图瓦片，在Leaflet地图上直观展示信号覆盖质量，并支持导出KML用于Google Earth等工具。

- 目标用户：通信网络优化工程师、路测数据分析人员
- 核心价值：将离散路测点数据快速可视化为连续信号覆盖热力图，辅助网络覆盖分析与优化决策

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 上传CSV、查看热力图、导出KML |

### 2.2 功能模块

1. **数据上传页**：CSV文件上传、数据预览、字段映射配置
2. **热力图展示页**：Leaflet地图+热力图瓦片叠加、指标切换（RSRP/SINR）、图例、缩放平移

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 数据上传页 | 文件上传区 | 拖拽或点击上传CSV文件，支持.csv格式 |
| 数据上传页 | 字段映射 | 自动识别经度、纬度、RSRP、SINR列，支持手动选择 |
| 数据上传页 | 数据预览 | 上传后展示前20行数据表格预览 |
| 数据上传页 | 插值参数配置 | IDW幂参数(1-5)、搜索半径、网格分辨率 |
| 热力图展示页 | Leaflet地图 | 底图展示+热力图瓦片叠加，支持缩放平移 |
| 热力图展示页 | 指标切换 | RSRP/SINR切换，颜色方案联动 |
| 热力图展示页 | 图例面板 | 颜色-数值对应图例 |
| 热力图展示页 | 路测点标注 | 可选显示原始路测点位置 |
| 热力图展示页 | KML导出 | 一键导出当前热力图为KML文件 |
| 热力图展示页 | 统计面板 | 数据点数量、RSRP/SINR均值/最大/最小统计 |

## 3. 核心流程

用户上传CSV → 系统解析并预览 → 用户确认字段映射和参数 → 后端执行IDW插值 → 生成热力图瓦片 → 前端Leaflet展示 → 用户切换指标/导出KML

```mermaid
    "F" --> "G[前端Leaflet展示热力图]"
    "G" --> "H{用户操作}"
    "H" --> "I[切换RSRP/SINR指标]"

```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝科技风（#0A1628 底色 + #00D4AA 青绿高亮 + #FF6B35 橙色告警）
- 辅助色：#1E293B 卡片背景，#334155 边框
- 按钮风格：圆角8px，主按钮实心青绿，次按钮描边
- 字体：思源黑体/Noto Sans SC，标题18-24px，正文13-14px
- 布局风格：左侧控制面板 + 右侧地图主区域，暗色主题
- 图标风格：线条图标（lucide-react）

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 数据上传页 | 文件上传区 | 虚线边框拖拽区、上传图标、文件名显示 |
| 数据上传页 | 字段映射 | 下拉选择器（经度/纬度/RSRP/SINR） |
| 数据上传页 | 数据预览 | 暗色表格，斑马纹行 |
| 数据上传页 | 参数配置 | 滑块+数值输入，幂参数/半径/分辨率 |
| 热力图展示页 | 地图主区域 | 全幅地图，左上角指标切换Tab |
| 热力图展示页 | 图例 | 右下角浮动面板，渐变色条+数值标注 |
| 热力图展示页 | 统计面板 | 左侧浮动卡片，关键统计指标 |
| 热力图展示页 | 工具栏 | 顶部工具栏：路测点开关、KML导出按钮 |

### 4.3 响应式

- 桌面优先设计（1920x1080为主）
- 地图区域自适应容器宽度
- 控制面板在小屏幕下可折叠
