## 1. 产品概述

VXLAN GPE 协议封装/解封装模拟器——基于 Python + Scapy 后端模拟以太帧的 VXLAN GPE 封装与解封装，前端以协议栈可视化方式展示各层报文结构及数据流转过程。

- 目标用户：网络工程师、协议开发者、网络专业学生
- 核心价值：直观理解 VXLAN GPE 协议封装机制，快速验证封装/解封装逻辑

## 2. 核心功能

### 2.1 功能模块

1. **封装模拟页**：输入原始以太帧参数，一键封装为 VXLAN GPE 报文，展示协议栈层级
2. **解封装模拟页**：输入 VXLAN GPE 报文，一键解封装恢复原始以太帧，展示逆向解包过程
3. **协议栈可视化**：以分层卡片/嵌套方式展示每层协议头字段及载荷

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 封装模拟页 | 帧输入区 | 输入源MAC、目的MAC、以太类型、载荷内容 |
| 封装模拟页 | 封装参数区 | 设置外层IP、UDP端口、VNI、下一个协议字段 |
| 封装模拟页 | 协议栈展示区 | 分层展示 Ethernet→IP→UDP→VXLAN GPE→原始Ethernet 的完整协议栈 |
| 解封装模拟页 | 报文输入区 | 输入十六进制 VXLAN GPE 报文或使用预设示例 |
| 解封装模拟页 | 解封装结果区 | 展示逐层剥离过程及恢复的原始以太帧 |
| 协议栈可视化 | 层级详情面板 | 点击某层高亮展示该层所有字段及偏移量 |

## 3. 核心流程

用户输入原始以太帧参数 → 后端 Scapy 构建 Ether 帧 → 依次封装 IP→UDP→VXLAN_GPE → 返回完整协议栈各层数据 → 前端渲染协议栈层级图。解封装为逆向流程：输入封装后报文 → Scapy 逐层解包 → 返回各层解析结果 → 前端展示解包过程。

```mermaid
flowchart TD
    "A[用户输入以太帧参数]" --> "B[后端 Scapy 构建 Ether 帧]"
    "B" --> "C[封装 IP 头]"
    "C" --> "D[封装 UDP 头]"
    "D" --> "E[封装 VXLAN GPE 头（含 Next Protocol）]"
    "E" --> "F[返回各层协议数据]"
    "F" --> "G[前端渲染协议栈可视化]"
    "H[用户输入 VXLAN GPE 报文]" --> "I[后端 Scapy 解析报文]"
    "I" --> "J[逐层剥离：VXLAN GPE→UDP→IP→Ether]"
    "J" --> "K[返回各层解析结果与原始帧]"
    "K" --> "L[前端展示解封装过程]"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深色科技感（深灰 #0f172a + 青色 #06b6d4 强调色）
- 按钮风格：圆角微光泽，hover 时发光效果
- 字体：Fira Code（等宽，用于报文字段）、Outfit（UI 文字）
- 布局风格：左侧输入面板 + 右侧协议栈可视化面板
- 图标风格：Lucide 线性图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 封装模拟页 | 帧输入区 | 输入框组，深色背景，等宽字体显示 MAC/IP |
| 封装模拟页 | 封装参数区 | VNI 选择器、Next Protocol 下拉、UDP 端口输入 |
| 封装模拟页 | 协议栈展示区 | 嵌套卡片，由外到内依次缩进，每层带颜色标识和字段表格 |
| 解封装模拟页 | 报文输入区 | 十六进制编辑器风格文本框、预设按钮 |
| 解封装模拟页 | 解封装结果区 | 动画逐层展开，显示剥离过程动画 |

### 4.3 响应式

桌面优先设计，协议栈可视化在大屏上完整展示嵌套层级，小屏可横向滚动。
