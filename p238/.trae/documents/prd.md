## 1. 产品概述

CoAP 资源监控面板是一个物联网（IoT）数据可视化应用，通过 CoAP 协议的 Observe 机制实时监控传感器资源（温度、湿度），并在前端以实时曲线和资源列表的形式呈现数据变化。

- 面向 IoT 开发者和测试人员，提供 CoAP 协议的直观可视化调试工具
- 演示 CoAP Observe 模式的完整工作流，帮助理解轻量级物联网协议的数据推送机制

## 2. 核心功能

### 2.1 功能模块

1. **仪表盘页面**：资源列表、实时曲线图、连接状态指示器

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 仪表盘 | 资源列表卡片 | 显示所有 CoAP 资源（温度/湿度）的 URI、最新值、单位、最后更新时间 |
| 仪表盘 | 实时曲线图 | 以时间轴为横轴展示各资源历史数据变化趋势，自动滚动，支持暂停/继续 |
| 仪表盘 | 连接状态指示器 | 显示 CoAP 服务器和 WebSocket 桥接的连接状态（已连接/断开） |
| 仪表盘 | 数据统计 | 显示每个资源的最大值、最小值、平均值 |

## 3. 核心流程

用户打开仪表盘页面 → 前端通过 WebSocket 连接后端 → 后端启动 CoAP 服务器并创建模拟传感器资源 → 后端 CoAP 客户端 Observe 这些资源 → 传感器每 2 秒产生新数据 → CoAP 服务器推送通知给观察者 → 后端通过 WebSocket 将数据转发到前端 → 前端更新曲线图和资源列表

```mermaid
    "CoAP 服务器推送 Observe 通知" --> "后端转发数据到 WebSocket"
    "后端转发数据到 WebSocket" --> "前端更新曲线图和资源列表"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深青色（Teal）#0D9488 + 琥珀色（Amber）#F59E0B 作为强调色
- 按钮风格：圆角微凸起，带细微阴影
- 字体：JetBrains Mono（数据展示）+ Noto Sans SC（界面文字）
- 布局风格：暗色主题，卡片式布局，顶部导航栏
- 图标风格：Lucide 线性图标

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 仪表盘 | 顶部导航栏 | 品牌标识、连接状态指示灯（绿色/红色闪烁）、时间显示 |
| 仪表盘 | 资源列表卡片 | 资源图标、URI路径、当前值（大字）、单位、最后更新时间、迷你趋势线 |
| 仪表盘 | 实时曲线图 | 多折线图、时间轴自动滚动、悬浮提示框、暂停/继续按钮、图例 |
| 仪表盘 | 数据统计条 | 最大值/最小值/平均值标签，带色彩标识 |

### 4.3 响应式设计

- 桌面优先设计，大屏下资源卡片横向排列，曲线图占据主区域
- 平板端资源卡片改为2列，曲线图自适应宽度
- 移动端单列布局，曲线图全宽
