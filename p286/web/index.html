📚 L2TPv3 协议说明

L2TPv3（Layer 2 Tunneling Protocol version 3, RFC3931）是二层隧道协议的第三版，支持对 Ethernet、VLAN 等二层帧的透明传输。

核心概念：

  • 隧道 (Tunnel)：在两个 L2TP 对端之间建立的传输通道，由 Tunnel ID 标识
  • 会话 (Session/伪线)：隧道内的逻辑连接，由 Session ID 标识，承载特定类型的伪线（如 Ethernet）
  • Cookie：会话级别的可选认证值（4 或 8 字节），用于数据包校验，防止误投递
  • 封装类型：UDP（支持 NAT 穿越）或 IP（更低开销）
📦 L2TPv3 数据包格式 (over IP)
+------------------+
| IP Header | IP Protocol = 115
+------------------+
| Session ID (4B) | 本地会话标识
+------------------+
| Cookie (0/4/8B) | 可选的会话认证
+------------------+
| L2-Specific HDR | 可选子层
+------------------+
| Ethernet Frame | 封装的二层帧
+------------------+
