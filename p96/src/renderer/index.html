认证检测说明

🔢 奇偶校验 (Parity Check)

验证GPS导航消息的6位奇偶校验位。错误率超过10%可能表明信号被篡改。

🕵️ SMU检测

检测欺骗消息单元(Spoofed Message Unit)。分析导航比特模式相似度和时序一致性。

🔐 NMA认证

导航消息认证(Navigation Message Authentication)。通过熵值分析检测伪造的认证数据。

⏱️ 时序一致性

验证子帧传输时间间隔。偏离6秒标准超过1秒视为异常。
