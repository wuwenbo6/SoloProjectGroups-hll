# 电路图识别系统 (Circuit OCR)

基于Python + PaddleOCR + 图论的电路图识别系统，可以自动识别电路图中的元件（电阻、电容、IC）和连线，生成SPICE网表。

## 功能特性

- 🔍 **元件识别**: 自动识别电阻、电容、IC等电路元件
- 🔗 **连线检测**: 使用图论算法分析元件间的连接关系
- 📝 **SPICE网表生成**: 自动生成可用于电路仿真的SPICE网表
- 🌐 **REST API**: 提供完整的HTTP API接口
- 💾 **数据持久化**: SQLite数据库存储识别记录
- 🎨 **可视化前端**: 精美的Web界面展示识别结果

## 技术栈

- **后端框架**: FastAPI
- **OCR引擎**: PaddleOCR
- **图像处理**: OpenCV
- **图论分析**: NetworkX
- **数据库**: SQLAlchemy + SQLite
- **前端**: 原生HTML/CSS/JavaScript

## 安装说明

1. 安装Python依赖:
```bash
pip install -r requirements.txt
```

2. 启动服务器:
```bash
python main.py
```

3. 访问Web界面:
```
http://localhost:8000
```

## API接口

### 识别电路图
```
POST /api/recognize
Content-Type: multipart/form-data

参数:
- file: 电路图图片文件

返回:
- components: 识别的元件列表
- connections: 元件连接关系
- spice_netlist: SPICE网表
- visualization_path: 可视化结果图片路径
```

### 获取识别记录
```
GET /api/records
```

### 获取单条记录
```
GET /api/records/{id}
```

### 下载SPICE网表
```
GET /api/netlist/{id}
```

### 健康检查
```
GET /health
```

## 支持的元件类型

- **RESISTOR (电阻)**: 通过平行线和矩形特征识别
- **CAPACITOR (电容)**: 通过曲线特征识别  
- **IC (集成电路)**: 通过多引脚矩形特征识别

## 使用说明

1. 打开浏览器访问 http://localhost:8000
2. 上传电路图图片（支持JPG/PNG/BMP格式）
3. 点击"开始识别"按钮
4. 查看识别结果:
   - **可视化**: 对比原图和识别结果
   - **元件列表**: 查看识别的元件详情
   - **连接图**: 元件连接关系可视化
   - **SPICE网表**: 生成的电路仿真网表

## 注意事项

- 首次运行会自动下载PaddleOCR模型
- 建议使用清晰的电路图图片以获得更好的识别效果
- 识别准确率取决于图片质量和元件绘制的规范性
