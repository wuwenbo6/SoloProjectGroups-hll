ASN 格式转换器

2字节 ASN (1-64511) ↔ 4字节 ASdot 表示法

支持批量输入，每行一个值

输入值后点击"转换"按钮

转换算法

ASN → ASdot: 高16位 = ASN / 65536
低16位 = ASN % 65536
结果 = "高.低"
ASdot → ASN: ASN = 高 × 65536 + 低
AS_TRANS (23456): 2字节 BGP 发言者遇到4字节 ASN 时
使用 AS_TRANS=23456 作为占位符
如: ASN 65538 → ASdot "1.2"
2字节表示: AS_TRANS (23456)
