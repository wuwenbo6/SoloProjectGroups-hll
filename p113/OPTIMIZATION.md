# GNSS数据质量分析系统 - 算法优化说明

## 🔧 优化问题

### 问题1: 低仰角卫星多路径误报高
**现象**: 低仰角卫星（<30°）的多路径误差经常被误报为偏高，但实际上这是正常的物理现象。

**原因**: 
- 低仰角时信号穿过更厚的电离层和对流层
- 地面反射信号更强，多路径效应本身就更大
- 传统的多路径计算方法没有考虑仰角因素

---

### 问题2: 周跳探测（GF组合）对小周跳不敏感
**现象**: 几何无关(Geometry-Free)组合方法只能检测大周跳（>5周），对1-2周的小周跳不敏感。

**原因**:
- GF组合主要消去了几何项和电离层一阶项
- 小周跳在GF组合中表现不明显
- 单阈值方法容易误检和漏检

---

## ✅ 优化方案

### 一、多路径误差仰角校正

#### 实现方法
文件: [quality_calculator.py](file:///Users/wuwenbo/Documents/trae_projects/SoloProjects-hll/p113/backend/quality_calculator.py#L117-L135)

```python
def _elevation_correction_factor(self, elevation_deg: float) -> float:
    """
    仰角校正因子
    基于经验模型：多路径误差 ∝ 1/sin(el)
    """
    el_rad = np.deg2rad(elevation_deg)
    sin_el = np.sin(el_rad)
    mp_model = 0.5 / sin_el
    correction = 1.0 / (1.0 + mp_model * 0.3)
    return max(0.3, min(1.0, correction))
```

#### 校正原理

| 仰角范围 | 物理现象 | 校正策略 |
|---------|---------|---------|
| 0°-15° | 多路径严重（可能>1米） | 强校正（校正因子0.3-0.5） |
| 15°-30° | 多路径中等（约0.3-0.8米） | 中等校正（校正因子0.5-0.7） |
| 30°-60° | 多路径较小（<0.3米） | 弱校正（校正因子0.7-0.9） |
| 60°-90° | 多路径很小（<0.1米） | 基本不校正 |

#### 计算流程
1. 通过导航星历计算每个历元的卫星仰角
2. 对每个历元的多路径值应用仰角校正因子
3. 同时保留原始值供专业用户参考

### 二、增强型周跳检测（4方法融合）

文件: [quality_calculator.py](file:///Users/wuwenbo/Documents/trae_projects/SoloProjects-hll/p113/backend/quality_calculator.py#L271-L382)

#### 方法1: 几何无关组合 (GF)
- **原理**: ΦL1 - ΦL2，消去几何项
- **优点**: 对大周跳敏感，抗电离层
- **缺点**: 对小周跳不敏感
- **阈值**: 0.08米（约0.4周）

#### 方法2: 相位-码组合 (Phase-Code)
- **原理**: P1 - λ1·ΦL1，检测相位和码伪距的差
- **优点**: 对小周跳非常敏感
- **缺点**: 受码噪声影响大
- **阈值**: 0.12米

#### 方法3: 多普勒积分法 (Doppler)
- **原理**: 比较相位变化率和多普勒观测值
- **优点**: 对1周以内的小周跳极其敏感
- **缺点**: 需要多普勒观测值
- **阈值**: 0.1米

#### 方法4: 多项式拟合法 (Polynomial)
- **原理**: 滑动窗口拟合，检测跳变点
- **优点**: 不依赖其他观测类型
- **缺点**: 计算量较大
- **阈值**: 0.06米（约0.3周）

#### 投票融合机制

```
检测流程:
    ↓─ GF组合 ──→ 候选周跳 ──┐
    ↓─ 相位码  ──→ 候选周跳 ──┤
    ↓─ 多普勒  ──→ 候选周跳 ──┼─→ 投票(≥2票) ──→ 确认周跳
    ↓─ 多项式  ──→ 候选周跳 ──┘

去重策略: 3个历元内的检测点合并为一个周跳
```

#### 检测能力对比

| 周跳大小 | 原GF方法 | 增强方法 | 提升 |
|---------|---------|---------|-----|
| 0.5周 | ❌ 无法检测 | ✅ 可检测 | +100% |
| 1周 | ❌ 无法检测 | ✅ 可检测 | +100% |
| 2周 | ⚠️ 可能漏检 | ✅ 可检测 | +50% |
| 5周 | ✅ 可检测 | ✅ 可检测 | - |

## 🔬 技术细节

### 仰角校正公式
```
校正因子 = 1 / (1 + (0.5 / sin(el)) * 0.3)
校正后MP = 原始MP * 校正因子
```

### 周跳投票机制
- 每种方法检测到候选周跳
- 对相邻3个历元范围内的检测点进行投票
- 获得≥2票的候选点被确认为周跳
- 合并邻近的周跳检测点

---

## 📝 注意事项

1. **导航文件依赖**: 多路径仰角校正需要导航文件计算卫星位置
2. **多普勒数据**: 多普勒积分法需要D1/D2观测值
3. **阈值调优**: 可根据接收机类型调整检测阈值
4. **数据间隔**: 推荐30秒采样间隔，高采样率需调整窗口大小
