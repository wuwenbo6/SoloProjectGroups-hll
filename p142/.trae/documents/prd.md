## 1. 产品概述

低轨卫星星座（LEO）Web模拟器是一款面向航天通信研究人员和学生的交互式仿真平台，通过Three.js 3D可视化展示星链样式的低轨卫星星座，模拟星间激光链路和用户地面终端接入，实时计算并展示传播延迟与多普勒频移。

- 目标用户：航天通信研究人员、网络工程学生、卫星通信爱好者
- 核心价值：将抽象的卫星网络概念转化为直观的3D可视化交互，降低理解门槛

## 2. 核心功能

### 2.1 功能模块

1. **3D星座可视化页面**：地球模型、卫星轨道、星间链路、用户终端
2. **仿真控制面板**：卫星数量、轨道参数、时间倍速、链路开关
3. **实时数据面板**：传播延迟、多普勒频移、链路状态、拓扑信息

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 3D星座可视化 | 地球渲染 | 带纹理贴图的3D地球模型，支持自转与缩放 |
| 3D星座可视化 | 卫星轨道渲染 | 多个轨道平面卫星分布，Walker星座构型 |
| 3D星座可视化 | 星间激光链路 | 卫星间动态连线，实时拓扑变化 |
| 3D星座可视化 | 用户地面终端 | 地面用户标记，接入链路可视化 |
| 控制面板 | 仿真参数 | 卫星数量、轨道高度、倾角调整 |
| 控制面板 | 时间控制 | 播放/暂停、倍速（1x~1000x）、时间戳 |
| 实时数据面板 | 延迟计算 | 星间链路传播延迟（ms）展示 |
| 实时数据面板 | 多普勒计算 | 多普勒频移（kHz）实时显示 |
| 实时数据面板 | 拓扑信息 | 网络节点度数、连接矩阵、拓扑变化事件 |

## 4. 用户界面设计

### 4.1 设计风格

- **设计主题**：深空科技风（Deep Space Tech）
- **主色调**：深空蓝 `#0a1628`、星银灰 `#1a2a4a`、激光青 `#00d4ff`
- **辅助色**：卫星绿 `#00ff88`、告警红 `#ff4466`、轨道紫 `#8866ff`
- **按钮风格**：玻璃拟态，圆角12px，半透明背景
- **字体**：主字体 `Orbitron`（科技感）、辅助字体 `Rajdhani`（数据展示）
- **布局风格**：全屏3D画布 + 悬浮控制面板，半透明玻璃卡片
- **图标风格**：Lucide图标，线条风格，科技感

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 3D星座可视化 | 地球渲染 | 深蓝色调地球，星点背景，自转动画 |
| 3D星座可视化 | 卫星渲染 | 绿色立方体卫星本体，青色激光链路连线 |
| 3D星座可视化 | 地面终端 | 红色标记点，脉冲扩散动画 |
| 控制面板 | 参数控制 | 玻璃拟态卡片，滑块/开关，实时响应 |
| 控制面板 | 时间控制 | 播放按钮、时间戳显示、倍速滑块 |
| 实时数据面板 | 数据展示 | 玻璃拟态卡片，大字号数值，动画过渡 |

### 4.3 响应式设计

- 桌面端优先（1440x900），适配1080p及以上
- 控制面板和数据面板固定在屏幕边缘
- 3D画布自适应窗口大小

### 4.4 3D场景指导

- **环境**：深空背景，星点粒子，大气辉光效果
- **光照**：平行光模拟太阳光照，环境光辅助
- **相机**：PerspectiveCamera，初始距离3倍地球半径，支持轨道控制器
- **动画**：地球自转（~0.1rad/s），卫星沿轨道运动，链路脉冲效果
- **后处理**：Bloom发光效果，突出卫星和链路
- **交互**：鼠标拖拽旋转、滚轮缩放、点击卫星显示详情
