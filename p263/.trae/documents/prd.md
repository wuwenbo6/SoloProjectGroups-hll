## 1. 产品概述

NB-IoT小区重选模拟器是一个面向通信工程师和学生的教学与仿真工具，模拟NB-IoT终端在多小区环境中测量邻区RSRP信号，并基于3GPP S准则判断是否执行小区重选。通过可视化界面实时展示服务小区与邻区参数、S准则计算过程及重选决策结果。

- 目标用户：通信工程师、无线网络优化人员、通信专业学生
- 核心价值：直观理解NB-IoT小区重选机制，降低学习3GPP协议的门槛

## 2. 核心功能

### 2.1 功能模块

1. **仿真控制台**：启动/暂停/重置仿真，调节仿真速度，配置终端移动轨迹
2. **小区参数面板**：展示服务小区和所有邻区的配置参数（PCI、频点、RSRP、Q_rxlevmin等）
3. **S准则计算可视化**：实时展示S_rxlev计算过程，高亮满足/不满足重选条件的邻区
4. **重选决策日志**：记录每次重选事件的完整过程（触发时间、源小区、目标小区、判决依据）

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主仿真页面 | 仿真控制台 | 启动/暂停/重置仿真，调节仿真步进速度，单步执行 |
| 主仿真页面 | 小区拓扑地图 | 可视化展示小区分布（服务小区+邻区），终端位置及移动轨迹，信号强度热力图 |
| 主仿真页面 | 小区参数面板 | 服务小区与邻区参数表格（PCI、EARFCN、RSRP、Q_rxlevmin、Q_rxlevminoffset、Q_hyst、Q_offset），S_rxlev计算值，重选状态指示 |
| 主仿真页面 | S准则计算面板 | 公式展示 S_rxlev = Q_rxlevmeas - (Q_rxlevmin + Q_rxlevminoffset) - P_compensation，各参数实时计算值，柱状图对比各邻区S值 |
| 主仿真页面 | 重选决策日志 | 时间戳、事件类型（测量/判决/重选）、源小区→目标小区、关键参数快照 |

## 3. 核心流程

用户启动仿真后，终端按设定轨迹移动，每一步测量当前服务小区和邻区的RSRP值（含路径损耗模拟），计算S准则值判断邻区是否满足驻留条件，再基于R准则排序确定是否重选到更优邻区，重选完成后更新服务小区并记录日志。

```mermaid
    "计算R值排序" --> "R_n > R_s 持续Treselection?"
    "R_n > R_s 持续Treselection?" -->|"是"| "执行小区重选"
    "R_n > R_s 持续Treselection?" -->|"否"| "保持当前服务小区"
    "执行小区重选" --> "更新服务小区"
    "更新服务小区" --> "记录重选日志"
    "记录重选日志" --> "终端移动一步"
    "保持当前服务小区" --> "终端移动一步"
    "邻区不满足驻留条件" --> "终端移动一步"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝（#0A1628）科技感背景，搭配青绿（#00E5A0）高亮与橙色（#FF6B35）告警
- 辅助色：深灰卡片（#131E33）、中灰文字（#8899AA）
- 按钮风格：圆角矩形（8px），主按钮青绿填充，次按钮描边
- 字体：Rajdhani（标题/数据展示）+ Source Sans 3（正文）
- 布局风格：左侧拓扑地图 + 右侧参数面板，底部日志滚动区域
- 图标风格：线性科技感图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|---------|
| 主仿真页面 | 仿真控制台 | 顶部工具栏：播放/暂停按钮、重置按钮、速度滑块、步进按钮 |
| 主仿真页面 | 小区拓扑地图 | 中央大区域：Canvas绘制六边形小区、终端图标、信号强度色彩编码、移动轨迹虚线 |
| 主仿真页面 | 小区参数面板 | 右侧面板：服务小区卡片（高亮边框）+ 邻区列表卡片，每项含参数条形图 |
| 主仿真页面 | S准则计算面板 | 右侧下方：公式展示区 + 各邻区S值柱状图（绿>0/红<0） |
| 主仿真页面 | 重选决策日志 | 底部折叠面板：时间线日志列表，重选事件高亮橙色标记 |

### 4.3 响应式

- 桌面优先设计，最小支持1280px宽度
- 低于1280px时右侧面板折叠为底部抽屉
- 触屏优化：地图区域支持捏合缩放和拖拽
