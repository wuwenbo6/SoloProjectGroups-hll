## 1. 产品概述
一阶Sigma-Delta调制器仿真Web应用，用于数字信号处理教学和研究，展示过采样和噪声整形原理
- 主要用途：教育演示、信号处理研究、工程学习
- 目标用户：电子工程学生、信号处理工程师、科研人员

## 2. 核心功能

### 2.1 功能模块
1. **主仿真页面**：参数配置、调制器仿真、结果可视化

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 主仿真页面 | 参数配置 | 设置正弦波频率、幅度、采样点数、过采样率 |
| 主仿真页面 | Sigma-Delta调制器 | 实现一阶Δ-Σ调制算法，输出二进制比特流 |
| 主仿真页面 | 时域波形图 | 显示输入正弦波、量化误差、输出比特流 |
| 主仿真页面 | 频谱分析 | FFT计算并绘制频谱图，展示噪声整形效果 |
| 主仿真页面 | SNR计算 | 自动计算输出信号的信噪比并显示结果 |

## 3. 核心流程
用户配置参数 → 点击运行仿真 → 执行Sigma-Delta调制 → 计算FFT和SNR → 渲染时域和频域图表

```mermaid
flowchart LR
    A["参数配置"] --> B["运行仿真"]
    B --> C["生成正弦波输入"]
    C --> D["Sigma-Delta调制"]
    D --> E["输出二进制流"]
    E --> F["FFT频谱分析"]
    F --> G["SNR计算"]
    G --> H["绘制图表"]
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深蓝色科技风格（#0f172a 背景，#3b82f6 主色，#10b981 强调色）
- 按钮风格：圆角矩形，悬停有轻微缩放和发光效果
- 字体：JetBrains Mono（代码/数据）、Inter（正文）
- 布局：卡片式布局，左右分栏（左侧参数配置，右侧图表展示）
- 视觉风格：深色科技主题，玻璃拟态效果，数据可视化为主

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI Elements |
|-----------|-------------|-------------|
| 主仿真页面 | 参数配置区 | 滑块、数字输入框、实时更新 |
| 主仿真页面 | 控制区 | 运行/重置按钮，状态指示器 |
| 主仿真页面 | 时域图表 | 多线条波形，可缩放交互 |
| 主仿真页面 | 频域图表 | 频谱柱状图，噪声基底标注 |
| 主仿真页面 | 数据面板 | SNR数值显示，统计信息卡片 |

### 4.3 响应式
- 桌面端：左右双栏布局，参数栏固定宽度320px
- 平板端：上下布局，参数区在上，图表区在下
- 移动端：单列滚动布局，触控优化的滑块和按钮
