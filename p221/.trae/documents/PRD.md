## 1. 产品概述

H.265 裸流分析器是一个纯前端 Web 应用，用于解析和可视化 H.265 (HEVC) 编码的裸流文件。用户可以上传 .hevc 格式文件，应用将自动解析 NAL 单元并展示帧类型分布和 GOP 结构，帮助视频开发人员快速分析码流结构。

- 核心用途：视频编码分析、码流调试、教育学习
- 目标用户：视频开发工程师、多媒体研究人员、学生

## 2. 核心功能

### 2.1 功能模块

1. **文件上传区**：支持拖拽和点击上传 .hevc 裸流文件
2. **NAL 解析引擎**：解析 VPS/SPS/PPS/IDR/P 等 NAL 单元类型
3. **帧类型分布图**：饼图展示各类型 NAL 单元占比
4. **GOP 结构可视化**：时间轴展示 GOP 序列和帧顺序
5. **详细信息面板**：显示文件统计和各 NAL 单元详情

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 主页面 | 文件上传区 | 拖拽上传、点击选择文件、文件大小显示 |
| 主页面 | 统计概览 | 总帧数、各类型计数、文件大小、GOP 数量 |
| 主页面 | 帧类型分布图 | 饼图展示 VPS/SPS/PPS/IDR/P/B 帧占比 |
| 主页面 | GOP 结构视图 | 横向时间轴，彩色标记各帧类型，可缩放 |
| 主页面 | NAL 单元列表 | 表格展示所有 NAL 单元，支持搜索和筛选 |

## 3. 核心流程

用户打开应用 → 上传 .hevc 文件 → 前端解析 NAL 单元 → 生成统计数据 → 渲染图表和可视化 → 用户交互查看详情

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深科技蓝 (#165DFF)，代表专业和技术感
- **辅助色**：成功绿 (#00B42A)、警告橙 (#FF7D00)、错误红 (#F53F3F)
- **背景**：深色主题 (#141414)，搭配渐变和玻璃拟态效果
- **按钮风格**：圆角 8px，悬停时有微妙的发光效果
- **字体**：JetBrains Mono（等宽，适合显示技术数据） + Inter（正文）
- **布局风格**：卡片式布局，信息分区清晰，网格化排列

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|-------------|
| 主页面 | 文件上传区 | 虚线边框、拖拽高亮、文件图标、上传动画 |
| 主页面 | 统计卡片 | 渐变背景、大数字、图标、微妙阴影 |
| 主页面 | 饼图区域 | Chart.js 渲染、悬停提示、图例说明 |
| 主页面 | GOP 时间轴 | 横向滚动、彩色方块、悬停详情、缩放控制 |
| 主页面 | NAL 列表 | 斑马纹表格、固定表头、类型标签、搜索框 |

### 4.3 响应式设计

- **桌面端优先**：四列统计卡片、图表并排展示
- **平板适配**：两列统计卡片、图表垂直排列
- **移动端**：单列布局、简化时间轴、触摸友好的交互元素

### 4.4 动效设计

- 文件上传时的脉冲动画
- 图表加载时的渐进式渲染
- 卡片悬停时的轻微上浮和阴影加深
- 时间轴滚动时的视差效果
- NAL 类型切换时的平滑过渡
