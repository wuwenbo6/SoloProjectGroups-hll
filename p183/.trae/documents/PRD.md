## 1. 产品概述

SPICE模型参数拟合Web应用，用户上传实测V/I曲线数据，后端使用最小二乘法拟合并提取SPICE模型参数（如二极管IS、N），前端可视化展示实测曲线与拟合曲线对比。

- 主要用途：电子工程师快速从实测数据中提取器件模型参数
- 目标用户：电路设计工程师、半导体测试人员
- 价值：自动化参数提取，提高建模效率

## 2. 核心功能

### 2.1 功能模块

1. **数据上传模块**：支持CSV/TXT格式的V/I曲线数据上传
2. **参数拟合模块**：使用最小二乘法拟合二极管SPICE模型参数
3. **结果展示模块**：图表展示实测曲线与拟合曲线对比
4. **参数展示模块**：显示拟合得到的模型参数值

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 主页面 | 数据上传区 | 拖拽或点击上传V/I数据文件，支持CSV格式预览 |
| 主页面 | 图表展示区 | ECharts绘制实测曲线与拟合曲线对比图 |
| 主页面 | 参数结果区 | 显示拟合得到的IS、N等参数值及拟合优度 |
| 主页面 | 示例数据区 | 提供示例数据一键加载功能 |

## 3. 核心流程

用户上传V/I数据文件 → 后端解析数据 → 执行最小二乘拟合算法 → 返回拟合参数和拟合曲线数据 → 前端渲染对比图表和参数结果

```mermaid

    A["用户上传数据"] --> B["后端解析CSV"]
    B --> C["最小二乘拟合"]
    C --> D["计算拟合参数"]
    D --> E["生成拟合曲线"]
    E --> F["前端展示对比图"]
    F --> G["显示参数结果"]
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝色系（#1e3a5f），体现工程专业性
- 辅助色：青绿色（#10b981）用于高亮和成功状态
- 按钮风格：圆角矩形，悬停有阴影效果
- 字体：使用Geist Sans作为主字体，搭配Fira Code展示数值
- 布局：卡片式布局，左侧数据区，右侧图表区
- 图标风格：简洁线性图标（lucide-react）

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 主页面 | 数据上传区 | 拖拽区域、文件选择按钮、数据预览表格 |
| 主页面 | 图表展示区 | ECharts折线图、图例、坐标轴标签 |
| 主页面 | 参数结果区 | 参数卡片、数值显示、拟合优度指标 |
| 主页面 | 操作区 | 拟合按钮、重置按钮、示例数据按钮 |

### 4.3 响应式

- 桌面端优先：左右分栏布局
- 平板端：上下堆叠布局
- 移动端：单列流式布局，图表自适应宽度
