# CANopen SDO Tool - USB-CAN适配器

一个基于Electron的CANopen SDO（服务数据对象）工具，通过USB-CAN适配器（模拟）读写对象字典，支持超过4字节的数据使用分块传输。

## 功能特性

- ✅ 设备检测：连接前检测可用的USB-CAN适配器
- ✅ 设备选择：支持从检测到的设备列表中选择
- ✅ 设备信息显示：显示设备名称、序列号、厂商ID
- ✅ USB-CAN适配器模拟连接
- ✅ SDO读取对象字典
- ✅ SDO写入对象字典
- ✅ 快速传输（≤4字节）
- ✅ 分块传输（>4字节）
- ✅ 分块超时重试（最多3次）
- ✅ Toggle位校验
- ✅ 多种数据格式支持（十六进制、文本、整数）
- ✅ CAN消息实时日志显示
- ✅ 传输进度条显示（含重试次数）
- ✅ 预设快速测试按钮

## 使用说明

### 连接适配器

1. 点击"连接适配器"按钮
2. 状态指示灯变为绿色，表示已连接
3. SDO操作按钮将被启用

### SDO读取

1. 输入节点ID（默认: 1）
2. 输入索引（十六进制，如: 2000）
3. 输入子索引（默认: 1）
4. 点击"读取SDO"按钮
5. 结果将显示在右侧面板

### SDO写入

1. 输入节点ID、索引、子索引
2. 选择数据格式（十六进制、文本、整数）
3. 输入要写入的数据
4. 点击"写入SDO"按钮

### 快速测试

应用提供了三个预设测试按钮：
- **设备类型** (0x1000:0): 读取4字节数据（快速传输）
- **设备名称** (0x1008:0): 读取长文本（分块传输）
- **长数据测试** (0x2000:1): 读取50字节数据（分块传输）

## SDO协议说明

### 快速传输（Expedited Transfer）

- 数据长度 ≤ 4字节
- 单帧完成传输
- 命令字节: 0x23 (写), 0x42/0x43 (读)

### 分块传输（Segmented Transfer）

- 数据长度 > 4字节才使用分块传输
- 多帧完成传输
- 初始帧指定数据总长度
- 后续每帧携带最多7字节数据
- 使用toggle位进行帧确认
- **分块超时重试**：每帧超时500ms自动重试，最多3次
- **Toggle位校验**：校验帧序列的toggle位，不匹配时触发重传

### COB-ID分配

- SDO发送（客户端→服务器）: 0x600 + Node ID
- SDO响应（服务器→客户端）: 0x580 + Node ID

## 模拟对象字典

应用模拟了以下对象字典条目：

| 索引    | 子索引 | 名称       | 数据长度 | 传输类型 |
|---------|--------|------------|----------|----------|
| 0x1000  | 0x00   | 设备类型   | 4字节    | 快速     |
| 0x1008  | 0x00   | 设备名称   | 13字节   | 分块     |
| 0x2000  | 任意   | 长数据区   | 50字节   | 分块     |
