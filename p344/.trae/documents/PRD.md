## 1. 产品概述

CBOR诊断表示法转换器是一个纯前端Web工具，用于将CBOR（Concise Binary Object Representation）二进制数据与人类可读的诊断表示法（Diagnostic Notation）之间进行双向转换。用户可以通过上传二进制文件或输入Hex字符串来解码CBOR数据，也可以将诊断表示法编码回CBOR二进制格式。

- 目标用户：物联网开发者、协议工程师、安全研究人员、CBOR数据调试人员
- 核心价值：提供无需安装的在线CBOR编解码工具，降低CBOR数据调试门槛

## 2. 核心功能

### 2.1 功能模块

1. **解码页面**：Hex输入/文件上传 → CBOR诊断表示法输出
2. **编码页面**：诊断表示法输入 → CBOR Hex/二进制输出

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 解码页面 | Hex输入区 | 支持粘贴Hex字符串，实时解析CBOR |
| 解码页面 | 文件上传区 | 支持拖拽/点击上传CBOR二进制文件 |
| 解码页面 | 诊断表示法输出区 | 显示格式化的诊断表示法，支持复制 |
| 解码页面 | 解析树视图 | 以树形结构展示CBOR数据层级 |
| 编码页面 | 诊断表示法输入区 | 输入诊断表示法文本 |
| 编码页面 | Hex输出区 | 显示编码后的Hex字符串，支持复制 |
| 编码页面 | 二进制下载 | 下载编码后的CBOR二进制文件 |

## 3. 核心流程

### 解码流程
1. 用户输入Hex字符串或上传CBOR二进制文件
2. 系统解析Hex为Uint8Array
3. CBOR解码器将二进制数据解码为诊断表示法
4. 同时生成树形视图展示数据结构
5. 用户可复制诊断表示法文本

### 编码流程
1. 用户输入诊断表示法文本
2. 系统将诊断表示法编码为CBOR二进制
3. 输出Hex字符串，用户可复制或下载二进制文件

## 4. 用户界面设计

### 4.1 设计风格

- **主题**：工业风/终端风格（暗色系），体现二进制/协议工具的技术感
- **主色**：深灰底色（#0d1117）+ 青绿色高亮（#00d4aa）+ 橙色辅助（#ff6b35）
- **字体**：代码区使用等宽字体 JetBrains Mono / Fira Code，UI文字使用 DM Sans
- **布局**：左右双栏布局，输入在左，输出在右
- **按钮风格**：圆角微光效果，hover时发光增强
- **图标**：使用lucide-react图标库

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 解码页面 | Hex输入区 | 深色textarea，等宽字体，行号显示，实时校验 |
| 解码页面 | 文件上传区 | 虚线拖拽区域，上传图标，文件名显示 |
| 解码页面 | 诊断表示法输出区 | 语法高亮，JSON风格格式化，一键复制按钮 |
| 解码页面 | 解析树视图 | 可折叠树形节点，类型标签，值预览 |
| 编码页面 | 诊断表示法输入区 | 深色textarea，语法提示 |
| 编码页面 | Hex输出区 | Hex字节高亮，一键复制，字节统计 |
| 编码页面 | 二进制下载 | 下载按钮，文件大小提示 |

### 4.3 响应式设计

- 桌面端优先，左右双栏布局
- 平板端自动切换为上下布局
- 移动端单列布局，输入输出切换显示

### 4.4 示例数据

提供预设示例数据，方便用户快速体验：
- 简单整数：`0x18 2a` → `42`
- 文本字符串：`0x64 68 65 6c 6c 6f` → `"hello"`
- 嵌套Map：`{1: "a", 2: {3: "b"}}` → 对应CBOR Hex
- 数组：`[1, 2, 3]` → 对应CBOR Hex
