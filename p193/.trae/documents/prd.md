## 1. 产品概述
IRIG 106 Chapter 10 数据包解析Web应用，用于上传和解析航空航天测试数据文件，识别并展示不同类型的数据包摘要信息。
- 主要用途：帮助测试工程师快速查看IRIG 106文件内容，识别TMATS、PCM、MIL-STD-1553等数据包类型
- 目标用户：航空航天测试工程师、数据分析人员
- 产品价值：无需专用软件即可快速预览IRIG 106文件结构和内容摘要

## 2. 核心功能

### 2.1 功能模块
1. **文件上传页面**：拖拽/点击上传IRIG 106 Chapter 10文件，显示上传进度
2. **解析结果页面**：展示文件级信息和数据包列表摘要
3. **数据包详情**：点击数据包查看详细内容

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 首页 | 文件上传区 | 支持拖拽和点击上传.ch10/.irig文件，显示文件大小、上传状态 |
| 首页 | 解析概览 | 上传完成后显示文件基本信息：总数据包数、各类型包数量统计 |
| 首页 | 数据包列表 | 表格展示所有数据包：序号、包类型、时间戳、数据包长度、状态 |
| 首页 | 包详情面板 | 点击列表项展开显示该数据包的详细字段和内容摘要 |
| 首页 | 过滤筛选 | 按数据包类型、时间范围筛选显示 |

## 3. 核心流程
用户上传IRIG 106文件 → 前端发送文件到后端 → 后端解析文件头和数据包结构 → 返回解析结果 → 前端展示数据包列表和统计信息 → 用户点击查看详情

```mermaid
flowchart LR
    A["用户上传文件"] --> B["前端校验文件格式"]
    B --> C["发送文件到后端API"]
    C --> D["后端解析IRIG 106文件结构"]
    D --> E["识别TMATS/PCM/1553包类型"]
    E --> F["生成数据包摘要"]
    F --> G["返回JSON结果到前端"]
    G --> H["前端展示列表和统计"]
    H --> I["用户点击查看详情"]
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深蓝（#0F172A）配科技蓝（#3B82F6），符合航空航天工程风格
- 辅助色：绿色（#10B981）表示成功，橙色（#F59E0B）表示警告
- 按钮风格：直角设计，轻微悬浮效果，边框2px
- 字体：JetBrains Mono等宽字体用于数据展示，Space Grotesk用于标题
- 布局：顶部导航 + 左侧上传区 + 右侧数据区的双栏布局
- 图标风格：Lucide线性图标，科技感强

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 首页 | 文件上传区 | 虚线边框拖放区域，悬浮高亮，文件图标动画 |
| 首页 | 统计卡片 | 四个圆角卡片展示：总包数、TMATS包、PCM包、1553包 |
| 首页 | 数据包列表 | 斑马纹表格，行悬浮高亮，时间戳格式化显示 |
| 首页 | 详情面板 | 从右侧滑入的抽屉式面板，键值对展示详细字段 |
| 首页 | 过滤器 | 标签式类型筛选，点击切换激活状态 |

### 4.3 响应式
- 桌面端：双栏布局，上传区和数据区并排
- 平板端：上下布局，上传区在上，数据区在下
- 移动端：单列布局，表格可横向滚动，详情改为底部弹窗

### 4.4 视觉动效
- 文件上传时的进度条动画
- 解析完成后统计数字的滚动动画
- 数据包行悬浮时的微平移效果
- 详情面板滑入的过渡动画
