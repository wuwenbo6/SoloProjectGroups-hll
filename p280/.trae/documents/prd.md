## 1. 产品概述

H.323 网守模拟器是一个用于模拟 H.323 协议中网守（Gatekeeper）RAS 消息交互的 Web 应用。它支持终端发现（GRQ→GCF）、终端注册（RRQ→RCF）和呼叫许可（ARQ→ACF）等核心 RAS 流程，帮助开发者理解和调试 H.323 信令协议。

- 目标用户：VoIP 开发者、网络工程师、H.323 协议学习和测试人员
- 核心价值：提供可视化的网守模拟环境，实时展示 RAS 消息交互过程和终端管理状态

## 2. 核心功能

### 2.1 用户角色

| 角色 | 说明 | 核心权限 |
|------|------|----------|
| 操作员 | 网守管理操作员 | 查看终端、发起/管理呼叫许可、配置网守参数 |
| 模拟终端 | H.323 终端模拟器 | 发送 GRQ/RRQ/ARQ 消息、接收响应 |

### 2.2 功能模块

1. **控制台页面**：网守状态总览、RAS 消息实时日志、快速操作面板
2. **终端管理页面**：已注册终端列表、终端详情、手动注册/注销
3. **呼叫许可页面**：呼叫请求列表、许可状态、带宽管理

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|

| 终端管理 | 终端列表 | 表格展示所有已注册终端，含终端ID、别名、信令地址、注册时间、状态 |
| 终端管理 | 终端注册表单 | 填写终端信息发送 RRQ 请求进行注册 |

## 3. 核心流程

### 3.1 终端发现流程
终端发送 GRQ（Gatekeeper Request）消息，网守响应 GCF（Gatekeeper Confirm）或 GRJ（Gatekeeper Reject），完成终端与网守的发现握手。

### 3.2 终端注册流程
终端发现网守后，发送 RRQ（Registration Request）消息，网守验证后响应 RCF（Registration Confirm）或 RRJ（Registration Reject），终端完成注册。

### 3.3 呼叫许可流程
注册终端发起呼叫前，发送 ARQ（Admission Request）消息请求呼叫许可，网守根据带宽和策略响应 ACF（Admission Confirm）或 ARJ（Admission Reject）。

```mermaid
    "终端" -->|"GRQ (Gatekeeper Request)"| "网守"
    "网守" -->|"GCF (Gatekeeper Confirm)"| "终端"
    "终端" -->|"RRQ (Registration Request)"| "网守"
    "网守" -->|"RCF (Registration Confirm)"| "终端"
    "终端A" -->|"ARQ (Admission Request)"| "网守"
    "网守" -->|"ACF (Admission Confirm)"| "终端A"
    "终端A" -->|"呼叫建立"| "终端B"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主题**：深色科技风，模拟网络监控中心
- **主色**：深蓝黑底 (#0a0e1a) + 青色高亮 (#00e5ff) + 绿色成功 (#00ff88)
- **辅色**：橙色警告 (#ff9100) + 红色拒绝 (#ff3d71)
- **按钮**：圆角微光感，hover 时发光效果
- **字体**：JetBrains Mono（代码/数据）+ Space Grotesk（标题/UI）
- **布局**：左侧导航 + 右侧内容区，卡片式模块
- **动效**：消息流入动画、状态切换脉冲、数据更新闪烁

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 控制台 | 网守状态卡片 | 深色卡片、图标+数值、脉冲动画指示在线状态 |
| 控制台 | RAS消息日志 | 深色终端风格日志区、彩色消息标签、自动滚动、方向箭头 |
| 控制台 | 快速操作面板 | 发光按钮组、表单输入、下拉选择 |
| 终端管理 | 终端列表 | 深色表格、行 hover 高亮、状态标签（绿色在线/灰色离线） |
| 终端管理 | 注册表单 | 模态弹窗、输入框+标签、提交按钮 |
| 呼叫许可 | 呼叫列表 | 卡片列表、方向指示（主叫→被叫）、状态徽章 |
| 呼叫许可 | 带宽仪表盘 | 环形进度图、数值标注、颜色渐变 |

### 4.3 响应式设计

- 桌面优先设计，最小宽度 1280px
- 侧边栏可折叠
- 表格支持横向滚动
