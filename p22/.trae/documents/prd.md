## 1. 产品概述

基于FFmpeg.wasm的浏览器端视频处理Web应用，用户可直接在本地进行视频格式转换、滤镜处理，无需上传到服务器，保护隐私且速度快。

- 核心价值：零服务器依赖、本地处理、隐私安全、操作简便
- 目标用户：需要快速处理视频的普通用户、开发者、内容创作者

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 普通用户 | 无需注册 | 使用所有视频处理功能 |

### 2.2 功能模块
1. **主处理页面**：文件上传、命令行输入、处理状态、结果下载
2. **快捷滤镜面板**：缩放、裁剪等常用滤镜快捷设置
3. **历史记录**：本地保存处理历史（可选）

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主页面 | 文件上传区 | 拖拽上传、点击选择文件、文件预览 |
| 主页面 | 命令行界面 | 模拟终端输入FFmpeg参数、快捷参数模板 |
| 主页面 | 滤镜面板 | 可视化设置缩放尺寸、裁剪区域 |
| 主页面 | 处理状态 | 实时进度显示、日志输出、处理时间 |
| 主页面 | 结果展示 | 视频预览、一键下载、格式信息 |

## 3. 核心流程

用户上传视频文件 → 输入或选择FFmpeg命令参数 → 配置滤镜选项 → 点击开始处理 → 浏览器本地执行FFmpeg.wasm → 显示处理进度和日志 → 预览处理结果 → 下载输出文件

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深色主题，深蓝紫色渐变（#1a1a2e → #16213e），科技感十足
- **强调色**：霓虹青色（#00d9ff），用于按钮、进度条、高亮
- **辅助色**：珊瑚橙（#ff6b6b）用于警告，薄荷绿（#4ecdc4）用于成功状态
- **按钮风格**：圆角8px，渐变背景，hover时有发光效果
- **字体**：JetBrains Mono（等宽字体，终端风格）+ Inter（界面文字）
- **布局风格**：卡片式布局，玻璃拟态效果（backdrop-filter）
- **图标风格**：简约线性图标，使用Lucide React

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 主页面 | 头部 | Logo、标题、简洁导航、渐变背景 |
| 主页面 | 上传区 | 虚线边框、拖拽高亮、文件图标、文件名显示 |
| 主页面 | 终端区 | 黑色背景、绿色光标、模拟终端输入、命令历史 |
| 主页面 | 滤镜面板 | 可折叠侧边栏、滑块控件、实时预览参数 |
| 主页面 | 状态区 | 进度条动画、日志滚动输出、状态指示灯 |
| 主页面 | 结果区 | 视频播放器、下载按钮、文件信息卡片 |

### 4.3 响应式
- Desktop-first设计，支持1280px以上宽屏
- 平板端（768px-1280px）：滤镜面板改为底部抽屉
- 移动端（<768px）：单列布局，简化界面，保留核心功能

### 4.4 动效设计
- 页面加载：元素渐入动画，staggered延迟效果
- 文件拖拽：上传区边框发光动画
- 处理进度：进度条流光效果
- 按钮交互：hover时scale+glow效果
- 终端输入：光标闪烁动画，打字机效果
