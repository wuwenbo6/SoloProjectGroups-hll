## 1. 产品概述

SMI-S 存储拓扑查看器 —— 通过 pywbem 连接 SMI-S Provider，枚举存储池、卷和掩码视图，以交互式拓扑图的形式直观展示存储资源的层级关系与映射关系。

- 解决存储管理员需要登录多个存储阵列管理界面的痛点，提供统一的可视化拓扑视图
- 目标用户：存储管理员、运维工程师

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 存储管理员 | 直接访问 | 连接 SMI-S Provider、浏览存储拓扑、查看设备详情 |

### 2.2 功能模块

1. **连接管理页**：SMI-S Provider 连接配置、连接状态监控
2. **拓扑总览页**：存储拓扑可视化、设备节点交互、层级关系展示
3. **设备详情页**：存储池/卷/掩码视图详细信息展示

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 连接管理页 | 连接配置表单 | 输入 SMI-S Provider 地址、端口、用户名、密码，支持保存配置 |
| 连接管理页 | 连接状态面板 | 显示当前连接状态、Provider 信息、枚举进度 |
| 拓扑总览页 | 拓扑图区域 | 交互式力导向图，展示 存储系统→存储池→卷→掩码视图 层级关系 |
| 拓扑总览页 | 图例与筛选 | 按设备类型筛选节点、图例说明 |
| 拓扑总览页 | 节点详情浮层 | 点击节点弹出设备属性面板 |
| 设备详情页 | 设备信息表格 | 存储池列表、卷列表、掩码视图列表，支持搜索与排序 |

## 3. 核心流程

用户打开应用 → 配置 SMI-S Provider 连接 → 系统通过 pywbem 枚举 CIM 类 → 提取存储池(StoragePool)、卷(StorageVolume)、掩码视图(MaskingView)及其关联 → 生成拓扑数据 → 前端渲染交互式拓扑图。

```mermaid
flowchart TD
    "A[用户配置 SMI-S 连接]" --> "B[后端 pywbem 连接 Provider]"
    "B" --> "C{连接成功？}"
    "C" -->|"是"| "D[枚举 CIM_StoragePool]"
    "C" -->|"否"| "E[显示连接错误]"
    "D" --> "F[枚举 CIM_StorageVolume]"
    "F" --> "G[枚举 CIM_MaskingView]"
    "G" --> "H[构建拓扑关系数据]"
    "H" --> "I[前端渲染拓扑图]"
    "I" --> "J[用户交互：点击/筛选/缩放]"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深色工业风（#0a0e17 深蓝黑底色 + #00d4ff 科技蓝高亮 + #ff6b35 橙色警示）
- 按钮风格：圆角微3D，深色背景 + 蓝色边框发光效果
- 字体：JetBrains Mono（代码/数据风格）+ Outfit（UI文字）
- 布局风格：左侧导航 + 主内容区，拓扑图全屏展示
- 图标风格：线性科技风图标（lucide-react）

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 连接管理页 | 连接配置表单 | 深色卡片式表单，输入框带蓝色聚焦边框发光，连接按钮带脉冲动画 |
| 拓扑总览页 | 拓扑图区域 | 深色画布背景，节点带发光光晕，连线带流动粒子动画，悬停放大效果 |
| 拓扑总览页 | 图例与筛选 | 左下角半透明面板，圆形色块图例，胶囊式筛选按钮 |
| 设备详情页 | 设备信息表格 | 深色表格，行悬停发光高亮，列头固定 |

### 4.3 响应式设计

- 桌面优先设计，拓扑图区域自适应全屏
- 移动端：表格横向滚动，拓扑图支持触摸缩放

### 4.4 3D场景指引

- 不适用
