关于 SCSI T10 PI

SCSI T10 PI (Protection Information) 是一种数据完整性标准，用于在存储设备传输过程中检测和防止数据损坏。

DIF (Data Integrity Field) 每个 512 字节扇区附加 8 字节保护信息：

  • Guard Tag (2字节): CRC16 校验和，用于检测数据损坏
  • Application Tag (2字节): 应用程序自定义标签
  • Reference Tag (4字节): 参考标签，通常为逻辑块地址 (LBA)
