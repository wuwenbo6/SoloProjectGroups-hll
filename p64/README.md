# 有限元应力分析系统

一个基于Web的有限元分析（FEA）应用，支持几何绘制、边界条件设置、应力计算和结果可视化。

## 功能特性

### 🎨 几何建模
- **矩形绘制**：在画布上拖拽绘制矩形
- **圆形绘制**：在画布上拖拽绘制圆形
- 实时显示几何尺寸信息

### 🔧 边界条件
- **固定约束**：指定边固定位移
- **集中力加载**：在指定边上施加集中力
- 支持X/Y方向力加载
- 可视化显示边界条件位置

### 📊 材料与网格
- 可配置弹性模量(E)和泊松比(ν)
- 支持5级网格细化
- 自动生成结构化网格

### 🚀 求解器
- 基于NumPy实现的2D弹性力学有限元求解器
- 支持平面应力分析
- 计算Von Mises等效应力
- 输出VTK格式结果文件

### 📈 结果可视化
- **云图显示**：应力/位移场彩色云图
- **多场量切换**：Von Mises、σ_xx、σ_yy、τ_xy、位移
- **显示模式**：线框、表面、带边线表面
- **变形放大**：可调变形放大系数
- **交互操作**：平移、缩放查看

### 💾 案例管理
- SQLite数据库存储仿真案例
- 案例列表查看和管理
- 支持保存、更新、删除操作

## 技术栈

### 后端
- **Flask** - Web框架
- **SQLAlchemy** - ORM数据库
- **NumPy** - 数值计算（有限元求解）
- **VTK** - 结果文件输出

### 前端
- **Vue 3** - 前端框架
- **Vite** - 构建工具
- **Canvas API** - 几何绘制
- **自定义渲染器** - 结果云图可视化

## 快速开始

### 启动后端服务

```bash
cd backend
python run.py
```

后端服务运行在 `http://localhost:8000`

### 启动前端服务

```bash
cd frontend
npm install
npm run dev
```

前端服务运行在 `http://localhost:3000`

## 使用说明

### 创建新仿真

1. 点击左侧"新建案例"按钮创建新案例
2. 在中间画布上选择绘制工具（矩形或圆形）
3. 在画布上拖拽绘制几何形状
4. 在边界条件区域添加约束和载荷
5. 配置材料属性和网格细化等级
6. 点击"开始计算"按钮运行仿真
7. 计算完成后查看应力云图

### 查看已有案例

1. 在左侧案例列表中点击案例
2. 案例信息自动加载
3. 可重新计算或查看已有结果
4. 点击案例进入详情页面

### 结果查看

- **显示模式**：切换线框/表面/带边线
- **场量选择**：切换显示不同的物理量
- **变形放大**：调整变形显示比例
- **平移缩放**：鼠标拖拽平移，滚轮缩放

## API接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/simulations` | 获取所有仿真案例 |
| GET | `/api/simulations/:id` | 获取单个案例详情 |
| POST | `/api/simulations` | 创建新案例 |
| PUT | `/api/simulations/:id` | 更新案例信息 |
| DELETE | `/api/simulations/:id` | 删除案例 |
| POST | `/api/simulations/:id/run` | 运行仿真计算 |
| GET | `/api/results/:filename` | 下载结果文件 |

## 有限元理论

本系统采用2D三角形单元求解线弹性问题：

- **单元类型**：3节点三角形常应变单元
- **本构模型**：线弹性各向同性材料
- **求解方法**：直接刚度法
- **边界条件**：Dirichlet（位移）和Neumann（力）边界

## 注意事项

1. 本系统使用NumPy实现有限元求解，未使用FEniCS库（避免复杂的环境配置）
2. 求解器为简化版本，适用于教学和演示目的
3. 网格细化等级建议不超过4（避免计算时间过长）
4. 结果文件保存在 `backend/results` 目录
