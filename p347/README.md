# 🔄 自动分层存储模拟器

一个基于 Python Flask 的自动分层存储模拟器，模拟 SSD/SAS/SATA 三层存储架构，数据块根据访问热度自动迁移。

## ✨ 功能特性

- **三层存储架构**: SSD (高性能) / SAS (平衡) / SATA (大容量)
- **热度算法**: 基于访问次数和时间衰减计算数据块热度
- **自动迁移**: 热数据自动升级到 SSD，冷数据自动降级到 SATA
- **实时可视化**: 前端展示容量分布、迁移统计和各层数据块
- **交互式控制**: 支持手动创建块、访问块、迁移块
- **自动模拟**: 支持自动模拟访问和自动分层

## 📊 核心算法

### 热度计算公式

```
热度 = (访问次数 / 最大访问次数) × 衰减因子
衰减因子 = max(0, 1 - 衰减率 × 距上次访问时间)
```

### 分层策略

- **热度 ≥ 0.7**: 热数据 → 迁移到 SSD
- **热度 ≤ 0.3**: 冷数据 → 迁移到 SATA
- **0.3 < 热度 < 0.7**: 温数据 → 保留在当前层

## 📡 API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/status` | 获取模拟器状态 |
| GET | `/api/blocks/<tier>` | 获取指定层的数据块 |
| POST | `/api/blocks` | 创建新数据块 |
| POST | `/api/access/<block_id>` | 访问指定数据块 |
| POST | `/api/migrate/<block_id>/<tier>` | 迁移数据块 |
| POST | `/api/auto-tier` | 执行自动分层 |
| POST | `/api/simulate-access` | 模拟随机访问 |
| POST | `/api/reset` | 重置模拟器 |
| POST | `/api/init-demo` | 初始化演示数据 |

## 🔧 配置参数

默认存储配置：
- SSD: 100 GB
- SAS: 500 GB
- SATA: 2000 GB

热度阈值：
- 高热阈值: 0.7
- 低热阈值: 0.3
- 衰减率: 0.001
