## 1. 产品概述

MODBUS到OPC UA映射配置工具是一个工业自动化领域的Web应用，用于解析MODBUS设备映射Excel文件，生成OPC UA服务器地址空间，并提供可视化的映射规则配置和节点浏览功能。

- **主要用途**：帮助工业自动化工程师快速配置MODBUS设备与OPC UA服务器的数据映射关系
- **解决问题**：简化传统OPC UA地址空间配置的复杂流程，提供可视化配置界面
- **目标用户**：工业自动化工程师、系统集成商、设备运维人员
- **产品价值**：提高配置效率，减少人工错误，实现设备数据的标准化接入

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 管理员 | 默认账号 | 完整的配置管理、文件上传、服务器控制权限 |

### 2.2 功能模块

1. **仪表盘**：服务器状态概览、设备统计、快捷操作
2. **映射文件管理**：Excel文件上传、解析预览、映射规则配置
3. **OPC UA节点浏览**：树形结构浏览地址空间、节点属性查看
4. **系统配置**：数据库配置、服务器参数设置、映射规则管理

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 仪表盘 | 状态卡片 | 显示OPC UA服务器运行状态、已连接设备数、节点总数 |
| 仪表盘 | 快捷操作 | 一键启动/停止服务器、快速上传映射文件 |
| 映射配置 | 文件上传 | 支持拖拽上传MODBUS设备映射Excel文件 |
| 映射配置 | 解析预览 | 表格展示Excel解析结果，支持编辑映射规则 |
| 映射配置 | 规则编辑器 | 配置MODBUS寄存器到OPC UA节点的映射关系 |
| 节点浏览器 | 树形导航 | 层级展示OPC UA地址空间节点 |
| 节点浏览器 | 属性面板 | 显示选中节点的详细属性（NodeId、数据类型、值等） |
| 系统设置 | 配置管理 | 数据库连接配置、服务器参数配置 |

## 3. 核心流程

用户上传MODBUS设备映射Excel文件 → 系统解析Excel并预览数据 → 用户配置映射规则 → 规则保存到数据库 → 生成OPC UA地址空间 → 启动OPC UA服务器 → 在线浏览节点

```mermaid
    E --> F["生成OPC UA地址空间"]
    F --> G["启动OPC UA服务器"]
    G --> H["在线浏览节点"]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深蓝色系（#1e3a5f），体现工业科技感
- **辅助色**：青色（#0ea5e9）用于交互元素，橙色（#f97316）用于警示
- **按钮风格**：圆角8px，悬停时有轻微上浮和阴影效果
- **字体**：使用JetBrains Mono作为代码字体，Inter作为界面字体
- **布局风格**：侧边导航 + 主内容区，卡片式模块布局
- **图标风格**：线性图标，使用Lucide图标库

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 仪表盘 | 状态卡片 | 渐变背景、图标动画、实时数据更新 |
| 映射配置 | 上传区域 | 拖拽高亮、文件类型限制、进度条 |
| 映射配置 | 数据表格 | 斑马纹、行悬停效果、可编辑单元格 |
| 节点浏览器 | 树形结构 | 展开/折叠动画、选中高亮、搜索过滤 |
| 系统设置 | 表单 | 标签左对齐、输入框边框高亮、验证提示 |

### 4.3 响应性
- 采用桌面端优先设计，适配1920×1080及以上分辨率
- 侧边栏在小屏幕可折叠
- 表格支持横向滚动
