## 1. 产品概述

MicroPython REPL 远程桥接工具 — 一个基于 Node.js + WebSocket 的全栈 Web 应用，桥接 MicroPython 设备的 UART 串口或 Telnet REPL，让开发者通过浏览器即可执行 Python 命令并实时查看输出。

- 解决开发者需要物理连接设备才能调试 MicroPython 的痛点，提供远程、实时的 REPL 交互体验
- 面向嵌入式开发者、IoT 工程师和硬件爱好者，降低 MicroPython 开发调试门槛

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 开发者 | 无需注册，直接使用 | 连接设备、执行命令、查看输出 |

### 2.2 功能模块

1. **连接管理页**：设备连接配置（UART/Telnet）、连接状态监控、设备列表管理
2. **REPL 终端页**：Python 命令输入、实时输出显示、命令历史、快捷操作

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 连接管理页 | 连接配置面板 | 选择连接方式（UART/Telnet）、填写串口路径/波特率或主机地址/端口、一键连接/断开 |
| 连接管理页 | 设备状态指示器 | 实时显示连接状态（已连接/断开/重连中）、设备信息展示 |
| 连接管理页 | 连接历史记录 | 记录最近使用的连接配置，支持快速重连 |
| REPL 终端页 | 终端输出区 | 实时流式显示 MicroPython REPL 输出，支持 ANSI 颜色渲染 |
| REPL 终端页 | 命令输入区 | 多行 Python 代码编辑、Ctrl+C 中断、Ctrl+D 软复位快捷键 |
| REPL 终端页 | 工具栏 | 清屏、发送文件、粘贴模式切换、常用快捷命令按钮 |

## 3. 核心流程

用户打开应用 → 选择连接方式（UART/Telnet）→ 配置连接参数 → 建立连接 → 进入 REPL 终端 → 输入 Python 命令 → 实时查看输出 → 断开连接

```mermaid
    "C" --> "|Telnet| E[配置主机地址和端口]"

    "H" --> "J[输入 Python 命令]"
    "J" --> "K[WebSocket 发送命令]"
    "K" --> "L[后端转发至设备]"

    "M" --> "N[WebSocket 推送输出]"

```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色终端风格（#0a0e17 深蓝黑底色 + #00ff88 荧光绿主强调色 + #0ea5e9 天蓝辅助色）
- **按钮风格**：圆角微光按钮，悬停时发光效果
- **字体**：终端区域使用 JetBrains Mono 等宽字体，UI 区域使用 Outfit 无衬线字体
- **布局风格**：左侧导航 + 右侧内容区，终端区域最大化
- **图标风格**：线条风格图标，搭配荧光绿高亮

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 连接管理页 | 连接配置面板 | 卡片式布局，深色背景配荧光绿边框，输入框深色半透明，连接按钮荧光绿发光 |
| 连接管理页 | 设备状态指示器 | 圆点指示灯（绿色已连接/红色断开/黄色重连中），状态文字标签 |
| 连接管理页 | 连接历史记录 | 列表卡片，悬停高亮，点击快速填充配置 |
| REPL 终端页 | 终端输出区 | 全宽深色终端模拟器，JetBrains Mono 字体，行号可选显示，ANSI 颜色支持 |
| REPL 终端页 | 命令输入区 | 底部固定输入框，>>> 提示符前缀，多行自动扩展，语法高亮 |
| REPL 终端页 | 工具栏 | 顶部图标工具栏，荧光绿悬停效果，工具提示 |

### 4.3 响应式设计

- 桌面优先设计，终端区域占据主要空间
- 移动端适配：连接配置折叠为抽屉式面板，终端区域全屏显示
- 触控优化：增大按钮点击区域，支持滑动手势切换面板

### 4.4 动效设计

- 连接建立时：脉冲光环动画
- 数据传输时：状态栏数据流动画
- 终端输出：打字机效果流式显示
- 断开连接时：淡出动画
