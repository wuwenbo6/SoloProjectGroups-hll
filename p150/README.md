# Windows安全日志监控器

一个基于Electron的Windows安全日志实时监控应用，用于追踪登录事件和进程创建等安全相关操作。

## 功能特性

- 🔐 **实时监控** - 实时订阅Windows安全日志
- 📊 **事件统计** - 可视化展示各类安全事件统计数据
- 🔍 **事件过滤** - 支持按事件ID和级别筛选日志
- 📋 **详细信息** - 点击事件可查看完整详情
- 🎨 **现代化界面** - 深色主题，优雅的用户界面
- 🖥️ **跨平台演示** - 非Windows系统自动进入演示模式

## 支持的安全事件

| 事件ID | 事件类型 | 说明 |
|--------|----------|------|
| 4624 | 登录成功 | 用户成功登录系统 |
| 4625 | 登录失败 | 用户登录尝试失败 |
| 4634 | 注销 | 用户会话被注销 |
| 4647 | 用户发起注销 | 用户主动发起注销 |
| 4688 | 进程创建 | 新进程被创建 |
| 4689 | 进程退出 | 进程退出 |

## 安装与运行

### 前置要求

- Node.js 16 或更高版本
- Windows系统（用于获取真实安全日志）
- 管理员权限（读取安全日志需要）

### 安装依赖

```bash
npm install
```

### 运行应用

```bash
npm start
```

### 开发模式

```bash
npm run dev
```

## 使用说明

1. **启动监控** - 点击"开始监控"按钮开始接收安全日志
2. **过滤事件** - 使用下拉菜单按事件ID或级别筛选
3. **查看详情** - 点击表格中的任意事件行查看完整信息
4. **清空日志** - 点击"清空日志"按钮清除所有已接收的事件
5. **停止监控** - 点击"停止监控"按钮暂停日志接收

## 注意事项

### Windows系统

1. **管理员权限**：读取安全日志需要以管理员身份运行应用
2. **安全策略**：确保系统安全日志已启用
3. **进程创建日志**：如需监控进程创建（事件ID 4688），需要在组策略中启用"审核进程创建"

### 非Windows系统

- 在macOS或Linux系统上运行时，应用会自动进入演示模式
- 演示模式会生成模拟的安全事件数据供测试界面功能

## 实现原理

### 日志采集

应用使用Windows内置的`wevtutil`命令行工具来查询和订阅安全日志：

- **历史日志**：使用 `wevtutil qe Security` 获取最近的安全事件
- **实时监控**：通过PowerShell循环查询实现实时更新
- **XML解析**：解析事件XML格式提取关键信息

### 事件处理流程

1. `wevtutil` 输出XML格式的事件数据
2. 主进程解析XML并提取事件字段
3. 通过IPC将事件发送到渲染进程
4. 渲染进程更新UI表格和统计数据
